# AWS Console Bookmarklets
Web Browser Bookmarklets for the AWS Console.
The bookmarklets available here are intended to further enhance the Web Console experience by adding additional functionality.

## List of Bookmarklets

- [S3 Downloader](/s3-downloader/README.md)

## Changelog
See the [Changelog](CHANGELOG.md) for a full history.

## Resources
The following resources may be used to create Bookmarklets.

[**Making Bookmarklets**](https://gist.github.com/caseywatts/c0cec1f89ccdb8b469b1) by [Casey Watts](https://github.com/caseywatts) is an excellent introduction to how Bookmarklets work with an example how to get started.

[**Bookmarklet Maker**](https://caiorss.github.io/bookmarklet-maker/) by [Caio Rordrigues](https://github.com/caiorss)  allows you to generate Javascript that is safe to use as a bookmark value.
The site also includes suggestions and tips how to get started.
